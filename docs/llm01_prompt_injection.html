2 — Attacker intent

  • Override safety rules / content filters
  • Extract the system prompt or internal config
  • Exfiltrate user session data or PII
  • Escalate privileges within connected systems
  • Execute unauthorized actions (send email, query DB)
  • Manipulate model output (bias, misinformation)
  • Hijack agentic pipelines / tool calls
  • Bypass evaluation logic (resume scoring, fraud checks)
  • Establish a persistent malicious persona
  • Chain injections across RAG / multi-turn context

3 — Detection patterns

P01 — Instruction override
  • Explicit commands to ignore/discard prior instructions
  • Phrases targeting "system prompt", "context", or "rules"
P02 — Role/persona hijack
  • "You are now X" / "act as X with no limits"
  • Fictional AI persona used to launder restrictions
P03 — Jailbreak framing
  • "For a story / roleplay / creative writing..."
  • Fictional wrapper around harmful instruction
P04 — Privilege escalation
  • Claims of special modes: "developer mode", "sudo", "admin override"
  • Fake system notifications granting elevated access
P05 — Data extraction
  • Requests to repeat/display the system prompt
  • Instruction to forward, export, or embed context in output
P06 — Hidden intent (indirect)
  • Instructions buried in files, URLs, or images
  • Semantic mismatch: benign surface, malicious payload
P07 — Obfuscation
  • Base64 / hex / ROT13 encoded instructions
  • Emoji substitution, Unicode lookalikes, leet-speak
P08 — Multilingual evasion
  • Attack payload in non-English language
  • Mixed-language instructions to split filter coverage
P09 — Payload splitting
  • Malicious instruction split across multiple turns or documents
  • Trigger word defined early, payload delivered later
P10 — Adversarial suffix
  • Appended nonsense token strings that shift model behavior
  • Statistically adversarial sequences bypassing classifiers
P11 — Context manipulation
  • Injecting false prior turns ("As you agreed earlier...")
  • Reframing model identity mid-conversation
P12 — Cross-modal injection
  • Instructions embedded in images, audio transcripts, or PDFs
  • Exploiting gap between what humans see vs. what model parses
